Widgets:
This theme has two sidebars, Inward and Outward. due to the nature of that split, i’ve had to disable the standard blogroll widget. I've created a 'Links List' widget as a replacement, that is included with this theme.

Validation:
This theme, including the Links List widget, is valid XHTML 1.0 Transitional, and CSS. However, there are a great many plugins and widgets that are not, and additionally, any invalid posts will cause pages to have validation errors.
