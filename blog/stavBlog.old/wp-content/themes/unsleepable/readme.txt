This is the readme file for the "unsleepable" theme by Ben Gray at http://www.openswitch.org


-----------INSTALLATION:--------------
The installation for this theme is identical to any other theme.  Just upload it to your wp-content/themes directory.



-----------HEADER GRAPHIC:---------------
To use this theme you have to create your own title graphic.  See the tutorial on how to do so here: (http://www.openswitch.org/2006/04/16/title-graphic-how-to/).



-----------PLUGINS:----------------
This theme assumes that you have these plugins installed and activated:
	Brian's latest comments (http://meidell.dk/archives/2004/09/12/brians-latest-comments/)
	
	Extended Live Archives (http://www.sonsofskadi.net/extended-live-archive/)
	
	Flickr RSS (http://eightface.com/wordpress/flickrrss/)
